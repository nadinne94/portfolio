Visão geral

Objetivo

Organizar dados públicos da Câmara dos Deputados em uma arquitetura analítica moderna, permitindo análises sobre proposições, deputados, partidos, tramitações, votações, eventos e presença parlamentar.

Problema

A API pública possui dados ricos, mas distribuídos em vários endpoints, com relações dependentes e estruturas diferentes. O projeto resolve isso por meio de ingestão, padronização, enriquecimento, modelagem dimensional e publicação para BI.

Resultado

O pipeline gera tabelas analíticas em Star Schema e disponibiliza os dados em uma camada de serving para consumo em Power BI, com dashboard público para exploração dos principais indicadores legislativos.
